# Ukázkové redesigny webů - Hradec Králové

Rychlé nástřely nového webu pro firmy v Hradci Králové. Statické HTML/CSS makety pro potřeby oslovení, ne finální produkt.

Živý náhled (GitHub Pages): https://j4manek.github.io/hradec-kralove-web-redesigny/

## Potvrzené případy (skutečně zastaralý web, se srovnáním před/po)

Ověřeno reálnými screenshoty (desktop i mobil), ne jen textovou analýzou.

| Firma | Obor | Složka |
|---|---|---|
| Zubní ordinace Břetislavova | Zubař | `01-zubni-ordinace-bretislavova/` (vč. `porovnani.html`) |
| Kadeřnictví V Kopečku | Kadeřnictví | `07-kadernictvi-v-kopecku/` (vč. `porovnani.html`) |

## Ukázky designu (aktuální web firmy je ve skutečnosti v pořádku)

Vznikly předtím, než jsme ověřili skutečné weby reálným screenshotem - weby těchto firem nejsou zastaralé. Zůstávají jako portfolio ukázka stylu, ne jako podklad pro oslovení.

| Firma | Obor | Složka |
|---|---|---|
| Autoservis Trumf | Autoservis | `02-autoservis-trumf/` |
| Autocentrum Trio | Autoservis / prodej | `03-autocentrum-trio/` |
| M Kadeřnictví | Kadeřnictví | `04-m-kadernictvi/` |
| Salon Petra Koppová | Kadeřnictví | `05-salon-koppova/` |
| M-DENT | Zubař | `06-mdent/` |

Každá složka obsahuje `index.html`, `sluzby.html`, `kontakt.html` a `styles.css` - statické stránky bez buildu, jdou otevřít přímo v prohlížeči.
